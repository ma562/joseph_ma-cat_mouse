About

The game involves a cat which is programmed to be an "AI" with the ability to move autonomously and automatically to chase the mouse (controlled by the user through push buttons). The Cat & Mouse game was finalized on a hand-held PCB assembly. Background music is played through an audio jack and the display screen is designed to be a selective menu with touch screen.

An STM32F091RCT6 ARM microcontroller with 256 KB Flash memory and 32KB SRAM was used to program the game. SPI (Serial Peripheral Interface) was used to send data between the microcontroller and LCD display. DAC (Digital to Analog converter) was used to convert and generate music. ADC (Analog to Digital converter) was used to communicate the touch screen data. Timers and interrupts were used to listen and respond to any instructions depending on what the user is doing (e.g. pressing a button in the menu). GPIO was used to receive input from push buttons.

The type of touch screen is 3.2" ILI9341 TFT LCD with breakout board from Adafruit, which has the display of 320*240 resolution. It is a four-wire resistive touchscreen and we wrote a simple driver for it to detect single touch locations.
